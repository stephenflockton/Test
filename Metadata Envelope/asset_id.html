  • Metadata Envelope

    The Metadata Envelope reproduces various data for search and indexing in the data repository

    • Asset

      The <asset> element is the root container element for a content item.

    • Metadata

      The <metadata> element is a container element for the normalised metadata of the content item; commonly referred to as the metadata envelope.

      • <common-meta>

        The <common-meta> element contains metadata that is common to all content types.

        • <content-type>

          This element contains the enumeration representing the content type of the content item.

        • <asset-id>

          This element contains the internal database identifer for the given content item.

        • <revision>

          This element contains the revision number for the given content item.

        • <revision-id>

          This element contains the unique identifier for a specific revision of a given content item.

      • <publication-meta>

        Element to contain publication metadata that is common to content types which are standalone publications. (Books, Chapters and Articles)

<asset-id>

This element contains the internal database identifer for the given content item.

This identifier is for internal database use only and must not be used by any external application.

Example

<m:asset-id>10__1088_1361-6552_aaa5ee</m:asset-id>

Derivation

This element is set automatically by the ingestion service automatically depending on the type of asset loaded:
Content Type Value
Article DOI (. and / replaced with _)
Issue ISSN_VolumeNumber_IssueNumber
Volume ISSN_VolumeNumber
Journal ISSN
Book ISBN
Chapter ISBN_ChapterNumber
