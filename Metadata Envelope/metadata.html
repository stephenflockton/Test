Metadata

The <metadata> element is a container element for the normalised metadata of the content item; commonly referred to as the metadata envelope.

This element and all child elements are in the “meta” name-space defined as
 xmlns:m="http://ns.iop.org/namespaces/meta"
It is made up of the following sub-elements
  • common-meta - The metadata common to all content types
  • Publication specific metadata
  • relationships - Listings of the relationships between content items
  • vocabularies - Taxonomy and vocabulary information
  • binaries - Listing of the binary assets (PDFs, Images etc) that are associated with a content item
