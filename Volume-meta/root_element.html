    Documentation for the IOP Publishing volume-meta file format for providing volume level metadata.

    • Root Element

      The root element for a volume-meta file is a <volume-meta> element.

    • primary-issn

      Capture the primary journal ISSN within <primary-issn>.

    • volume

      Capture the volume number in the <volume> element.

    • publication-date

      Capture the date the volume is published in <publication-date>.

    • conference-volume-version

      Capture the conference publication status in <conference-volume-version>.

    • ***
    • conference-volume-display-name

      Capture the information to be displayed about the conference title within the <conference-volume-display-name>.

    • conference

      Capture the conference metadata elements within a <conference> element.
