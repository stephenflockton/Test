Issue status

The current status of the issue is captured in <issue-status>.

/issue-meta/issue-status
There are only two allowed values for <issue-status>:
  • open
  • closed
Use “open” when more articles might be added to the issue; use “closed” when there are no more articles to add to the issue.

For Conference Series titles, the <issue-status> is always “closed”.

Example

<issue-status>closed</issue-status>
