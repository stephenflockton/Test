Disp formula

Use <disp-formula> to capture numbered and unnumbered formula which are set apart from the text.

//p/disp-forumla

Though <disp-formula> is used for formula to be displayed apart from the text it is captured as part of the <p> element which contains the preceeding and proceeding text. The preceeding <p> must not be closed and a new <p> opened to capture <disp-formula> in.

<disp-formula> must be given an @id attribute. If <disp-formula> is numbered this is captured using <label>.

The image of the formula is captured in a closed <graphic> element with the image file name referenced via the @xlink:href attribute. The xlink namespace must also be attached to <graphic>. <inline-graphic> must not be used.
