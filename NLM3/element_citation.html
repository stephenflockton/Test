Element citation

Capture the reference data elements in an <element-citation> element.

/article/back/ref-list/ref/element-citation

Each <element-citation> must be given a @publication-type. The allowed values are given in this table

The only allowed children of <element-citation> allowed are as follows:
  • <person-group>
  • <year>
  • <article-title>
  • <source>
  • <volume>
  • <issue>
  • <fpage>
  • <lpage>
  • <page-range>
  • <elocation-id>
  • <pub-id>
  • <edition>
  • <series>
  • <publisher-loc>
  • <publisher-name>
  • <institution>
  • <comment>
  • <conf-name>
  • <conf-date>
  • <conf-loc>
  • <patent>

Use <person-group> to capture the contributors' information. Each <person-group> must have a @person-group-type attribute with the valur either “author” or “editor”. Capture all contributors of each type in one <person-group>.

For individual authors and editors capture each contributor using <surname> and <given-names> as children of <name>. Capture name suffixes such as “Jr” using <suffix> not as part of <surname>. Capture collaboations using <collab>. If “et al” is included as part of the contributors capture using an empty <etal> element

Capture the citation year using <year> using a 4 digit format “yyyy”. Other date elements are not captured.

If only a page number is supplied capture it using <fpage>. If a last page is also supplied capture it using <lpage> and capture the page range using <page-range>, separating the first and last pages with a hyphen.

If an article number is supplied capture it using <elocation-id>. <elocation-id> must be give the attribute @content-typewith the value “artnum”.

Capture the article DOI using a <pub-id> element with the attribute @pub-id-type with the value “doi”. Do not capture a DOI using an <ext-link> element.

Only the following data should be captured for an <element-citation> element with the @publication-type=“thesis” unless otherwise specified.

Capture the thesis title in <article-title>. Capture “PhD Thesis” in <source>. Capture the university or instituton name in <institution>.

The <ext-link> element must be a child of <comment>.
