  • Books

    Documentation for the IOP Publishing specific implementation of BITS 2.0 for book content

    • Root element

      The root element for a book is a <book> element.

    • Book Meta

      <book-meta >is a mandatory element for all books as a container for the book level metadata.

    • Front Matter

      Capture the front matter information for a book inside a <front-matter> element.

      • Front Matter Part

        <front-matter-part>is a mandatory element for all <front-matter> as a container for the front-matter metadata.

        • Front Matter Title

          Capture the title of the front matter section in a <title> element inside a <title-group> element.

        • Front Matter Contributors

          Front matter contributor information is captured in a <contrib-group> element.

        • Page Numbers

          Capture first and last page numbers for a chapter in fpage and lpage elements respectively.

      • Dedication

        Capture book dedication information in a <dedication> element.

      • Preface
      • Acknowledgements

        Capture the book acknowledgements in a <ack> element.

      • Full Biographies

        Capture full author biographies in a <bio> element.

      • Glossary

        Capture glossary information inside a <glossary> element.

      • Other Front Matter

        Capture non-standard prelim sections in a <front-matter-part> element.

    • Chapters

      All of a books chapters must be captured in seperate <book-part > elements within a <book-body> element.

    • Back Matter

      Capture the book back matter inside a <book-back> element.
