Glossary

Capture glossary information inside a <glossary> element.

If the glossary has a title capture that information inside a <title> element.

The content of the glossary must be captured in a <def-list> element. This element is made up of one or more <def-item> elements which in turn contain a <term> element which contains the term being defined and a <def> element which contains the definition of the term.
