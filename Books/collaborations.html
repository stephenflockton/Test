  • Books

    Documentation for the IOP Publishing specific implementation of BITS 2.0 for book content

    • Root element

      The root element for a book is a <book> element.

    • Book Meta

      <book-meta >is a mandatory element for all books as a container for the book level metadata.

      • Book ID

        All books must contain a <book-id> element containing the book level DOI

      • Book Title

        Capture the book title in a <book-title> element inside a <book-title-group> element.

      • Subject Group

        All books must contain a <subj-group> element containing subject categorisation information.

      • Contributors

        Contributor information for a book is captured in <contrib-group> elements.

        • Authors and Editors

          Capture each author or editor contributor in a <contrib> element child of <contrib-group>

        • Collaborations

          Capture a collaboration group in a <contrib> element child of <contrib-group>

        • Contrib Xrefs

          Contributors with any associated affiliation, footnote or biography information must have that information explictly linked using an <xref> element.

        • Affiliations

          Capture affiliations in <aff> elements.

        • Biographies

          Capture contributor biographies in a <bio> element

      • Publication Date

        A Publication date for the book must be captured in a <pub-date> element

      • ISSN

        Capture ISSN information in a <issn> element

      • ISBN

        Capture ISBN information in a <ISBN> element

      • Publisher

        Capture Publisher information in a <publisher> element.

      • Cover Files

        Capture cover file images in <supplementary-material> elements

      • Supplementary Data

      • Copyright and Licenses

        Capture Copyright and License information inside a <permissions> element.

      • Self-uris

        Capture the filenames of the PDF, EPUB and MOBI representations of the book in <self-uri> elements.

      • Abstract

        Capture Abstracts in a <abstract> element.

      • Counts

        Capture the page count of the book using a <book-page-count > element as a child of <counts>.

      • Custom-Meta

        Capture Custom Metadata in a< custom-meta-group> element.

      • Notes

        Capture notes in a <notes> element.

    • Front Matter

      Capture the front matter information for a book inside a <front-matter> element.

    • Chapters

      All of a books chapters must be captured in seperate <book-part > elements within a <book-body> element.

Collaborations

Capture a collaboration group in a <contrib> element child of <contrib-group>

Each <contrib> must have a @contrib-type attribute. The only allowed values for this attribute are “author” or “editor”.

Each <contrib> representing a collaboration group must have a child <collab> element containing the name of the collaboration group.

If a collaboration group has any associated affiliation, footnote or biography information those must be explictly linked using an <xref> element. See Contrib Xrefs.
